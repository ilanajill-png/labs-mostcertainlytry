# Lauren Gurgiolo Media Rights Pass

Started: 2026-05-29

Purpose: identify usable media for the Lauren Gurgiolo public portfolio and tag each item by practical rights status.

This is not legal advice. It is a publishing triage: what can be reused now with attribution, what can be embedded from official platforms, what needs explicit permission, and what should only be linked as reference.

## Status Key

- `safe to use`: open license found. Can copy into the site if attribution/license terms are followed.
- `safe to embed`: official/platform embed available. Embed the player or post; do not download/rehost the media.
- `needs permission`: public/official asset found, but no reusable license found. Ask rights holder before copying, cropping, or rehosting.
- `reference only`: link to it for research/context; do not copy or embed without more confirmation.

## Best Immediate Media Choices

1. Use the Wikimedia Okkervil River performance photo as the only clearly reusable static image.
2. Add official Bandcamp embeds for the main Dialtones releases instead of copying album art.
3. Add official YouTube embeds for the Dialtones videos instead of downloading video/stills.
4. Link press photos/articles rather than rehosting them.
5. Ask Lauren/Dialtones, David Lovas, Sahara Smith, and Paco Proano for explicit permission for the strongest still images and cover art.

### Use Now

- Keep the Wikimedia CC BY 2.0 photo as the main static image with visible attribution.
- Add a `Listen` section with Bandcamp embeds for `Static Sky`, `Faces`, `The Dialtones`, `Silent City`, and `Calculated Carelessness`.
- Add a `Watch` section with YouTube embeds for `Love to Live Official Video`, `Monk Official Video`, `Dodo`, and `Jumbotron GuitarSolo RedRocks`.
- Add `Room of Resonance` as a Vimeo embed in the installation section.

### Ask Permission

- David Lovas: official Dialtones photo set.
- Sahara Smith: Dialtones EPK band photo.
- Paco Proano: Static Sky cover art and any related artwork.
- Lauren/Dialtones: confirmation that official site/EPK images can be used on the public tribute/archive page.
- Austin Chronicle / named photographers: any press/performance photos worth reusing beyond a source link.

### Avoid For Now

- Downloading or rehosting YouTube, Vimeo, Spotify, SoundCloud, or Bandcamp media.
- Copying Noisy Ghost PR images directly.
- Pulling photos from Instagram/Facebook/TikTok without explicit permission or a platform embed.
