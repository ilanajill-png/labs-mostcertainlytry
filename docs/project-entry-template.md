# Project Entry Template

Use this shape for new Labs entries. The JSON object powers the public cards, filters, and table. The Markdown file can hold rough notes first, then become a polished case study.

## JSON Object

```json
{
  "id": "short-url-safe-name",
  "title": "Readable Project Title",
  "category": "Case Study",
  "type": "Workflow Design",
  "status": "Draft",
  "date": "2026-05-25",
  "featured": false,
  "tools": ["Tool One", "Tool Two"],
  "summary": "One or two sentences explaining what this is.",
  "outcome": "What changed, shipped, improved, or was learned.",
  "url": "projects/short-url-safe-name.html",
  "source": "docs/short-url-safe-name.md"
}
```

## Suggested Categories

- Case Study
- Build Log
- Workflow
- Research
- Experiment
- Template

```md
# Project Title

What problem or idea started this?

What was made and how?

```
