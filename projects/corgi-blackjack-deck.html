Design Notes

Standard structure

Each card keeps the familiar corner rank, suit icon, red/black color split, and pip placement so it still reads like a normal deck.

Corgi system

Number cards use a soft corgi watermark. Aces and face cards use larger Texas corgi portraits with cowboy hats, suit-color bandanas, and lone star details.

Game ready

The individual SVG files can be used directly in the blackjack picker or any static HTML prototype without raster export.
